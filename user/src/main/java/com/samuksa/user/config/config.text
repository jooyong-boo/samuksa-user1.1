특정 도메인 없이 공유하는 config를 모아놓은 패키지

RestTemplate
 내부에서 http 요청할때 사용
 현재는 사용부분이 없어 빈에 등록하지않음
 기본적인 타임아웃 제한만 설정하였다.
 만약 타임아웃을 제한하지않아 응답이 오지않는다면 모든 api가 대기상태에 빠질수있고
 쓰레드가 남아있지 않게될것이다

SwaggerConfig
 프론트에서 api를 적절하게 사용하기위해 제공하는 문서
 각각 필요한 도메인을 패키지별로 나누어서 그룹을 따로 묶지는않음

WebMvcConfig
 서로다른 서버환경에서 자원을 공유할때 cors를 설정해야한다
 앞으로 프로젝트는 다른 서버와 공유하며 관리할것이다.
